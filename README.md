# 🛍️ 이커머스 웹 플랫폼

---

## 📌 프로젝트 소개

**이커머스 웹 플랫폼**은 사용자가 상품을 검색하고 장바구니에 담아 결제할 수 있는 쇼핑몰 웹 서비스입니다.  
6인 팀으로 협업하며 ERD 설계부터 사용자 인증, 결제 기능까지 전반적인 시스템을 직접 구현했습니다.

## ✨ 주요 기능

- 🔐 소셜 로그인 (Kakao, Naver API)

- 🗃️ MyBatis를 통한 DB 연동
